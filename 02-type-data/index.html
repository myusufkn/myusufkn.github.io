The Concept of Data Type

Dalam pemrograman, tipe data adalah konsep penting.

Untuk dapat mengoperasikan variabel, penting untuk mengetahui sesuatu tentang tipenya.

Tanpa tipe data, komputer tidak dapat menyelesaikan ini dengan aman :

Syintax

let x = 19 + "Honda";              // 19Honda
let x = "12" + "Honda";            // 12Honda
let x = 10 + 7 + "Honda";          // 17Honda
let x = "Honda" + 11 + 3;          // Honda113
(karena awalnya string, maka yang di belakang dianggap string.)

Note : Saat menggunakan angka dan string, JS akan memperlakukan angka sebagai string.

JavaScript Number

Semua nomor JavaScript disimpan sebagai angka desimal (floating point) angka yang dapat ditampung JS hanya 64 Bit.

JavaScript mendukung number notation, defaultnya ada basis 10, JavaScript juga mendukung hexsadecimal, binary, dan octal.

Penulisan :

  • Hexsadecimal  : 0xff // diawali dengan nol dan x
  • Binary              : 0b10101  // diawali dengan nol dan b
  • Octal                : 0o10   // diawali dengan nol dan o

Angka dapat ditulis dengan, atau tanpa desimal :

Decimal

  • Bilangan Decimal tidak akurat.
  • Maksimal 17 digit di belakakng koma.

Boolean

Sebuah tipe data yang digunakan untuk merepresentasikan logika True atau False.

Bisa digunakan pada statemen pengkondisian, untuk menentukan aksi yang berbeda dan mengatur alur kendali program.

Walaupun type data boolean merupakan type data yang paling sedernaha, tapi type data boolean akan bayak digunakan di berbagai fitur pemograman.

Exsample

let x = 7;
let y = 13;
let z = 22;

let a = x == z;           
let b = z > y;

JavaScript Function

Function JavaScript adalah blok kode yang dirancang untuk melakukan tugas tertentu.

Function JavaScript dijalankan ketika "sesuatu" memanggilnya.

JavaScript Funtion Syintax

Function JavaScript didefinisikan dengan kata kunci Function, diikuti dengan nama, diikuti dengan tanda kurung ().

Nama Function dapat berisi huruf, angka, garis bawah, dan tanda dolar (aturan yang sama dengan variabel).

Tanda kurung dapat menyertakan nama parameter yang dipisahkan dengan koma:
(parameter1, parameter2, ...)

Kode yang akan dieksekusi, oleh fungsi, ditempatkan di dalam kurung kurawal: {}

Syintax

<script>
function name(parameter1, parameter2, parameter3) {
// code to be executed
}
</script>

Parameter fungsi tercantum di dalam tanda kurung () dalam definisi fungsi.

Argumen fungsi adalah nilai yang diterima oleh fungsi saat dipanggil.

Di dalam fungsi, argumen (parameter) berlaku sebagai variabel lokal.

Exsample

<script>
function kencuss(x1, x2) {
return x1 * x2;
}

document.getElementById("demo").innerHTML = kencuss(12, 37);
</script>

Output :
