JavaScript Operator Aritmatika

Operator Desciption

- Pengurangan

Operator precedence

Urutan operasi aritmatika :

No. Operator Description
1 ( ) Kurung
2 ** Pangkat
2 * Kali
2 / Bagi

3 - Kurang

Operators Comparison

Hasil akhir mengasilkan Bolean true or false.

Operator Description
== Sama dengan
!= Tidak sama dengan
=== Strict sama dengan
!== Strict tidak sama dengan
< Lebih besar dari
> Lebih kecil dari
<= Lebih besar sama dengan
>= Lebih kecil sama dengan

Exsample

var x = 7;

x == 7;              // true
x == "7";            // true
x == 578;            // false

x != 7;              // false
x != "7";            // false
x != 183;            // true

x === 7;             // true
x === "7";           // false
x === 94;            // false

x !== 72;            // true
x !== 7;             // false
x !== "7";           // true

x > 7;               // false
x > 10;              // true
x > "826";           // true

x < 7;               // false
x < 3;               // true
x < "68";            // false

x >= 7;              // false
x >= 49;             // true
x >= "6";            // true

x <= 1;              // true
x <= 5;              // true
x <= "68";           // false

String and Number

Syintax

let x = 19 + "7";                  // 197
let x = 19 + "Honda";              // 19Honda
let x = "12" + "Honda";            // 12Honda
let x = 10 + 7 + "Honda";          // 17Honda
let x = "Honda" + 11 + 3;          // Honda113
